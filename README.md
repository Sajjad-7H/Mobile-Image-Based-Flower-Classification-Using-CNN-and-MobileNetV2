🌸 Mobile Image-Based Flower Classification using CNN & MobileNetV2

This project focuses on building a deep learning-based image classification system that can identify different types of flowers from images captured using mobile devices.

Two models are implemented and compared:

A Custom Convolutional Neural Network (CNN)
A MobileNetV2 Transfer Learning Model

The goal is to analyze performance differences and achieve higher accuracy using transfer learning.

🚀 Key Features
Deep Learning Models
Comparative analysis between Custom CNN and MobileNetV2
Transfer Learning
Uses pre-trained MobileNetV2 to improve accuracy and reduce training time
Image Preprocessing
Resize images to 224 × 224
Normalize pixel values
Apply data augmentation:
Rotation
Zoom
Horizontal Flip
Performance Insight
MobileNetV2 outperforms the Custom CNN model in terms of accuracy
📊 Dataset Overview

The dataset contains images of 6 different flower categories:

🌺 Bougainvillea (Baganbilash)
🌸 Catharanthus roseus (Noyontara)
🌼 Dianthus chinensis (Dianthus)
🌹 Hibiscus (Joba)
🌻 Marigold (Genda)
🌷 Petunia (Petunia)
